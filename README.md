A package for angular differential imaging (ADI) along with its variants, such as reference differential imaging (RDI) and spectral differential imaging (SDI).

## Installation

From the Julia REPL

```julia
julia> ]

(1.3) pkg> add https://github.com/juliahci/ADI.jl
```
